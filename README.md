# 📊 Dashboard de Vendas por Estado no Brasil - Tableau

## 📌 Sobre o Projeto

Este projeto apresenta uma análise de vendas por estado no Brasil com foco em identificar oportunidades de crescimento, regiões mais lucrativas e pontos de prejuízo.

## 🎯 Objetivo

Analisar o desempenho de vendas por estado para apoiar decisões estratégicas, como:

* Expansão de mercado
* Redução de prejuízos
* Aumento de faturamento

## 🛠️ Ferramentas Utilizadas

* Tableau
* Excel (base de dados)

## 📷 Visual do Dashboard

![Dashboard](dashboard.png)

## 📈 Principais Insights

* A região Sudeste concentra o maior faturamento, sendo estratégica para o negócio
* Estados do Nordeste apresentam prejuízo, indicando necessidade de revisão de estratégia ou custos
* A região Sul apresenta crescimento consistente e potencial de expansão

## 💼 Aplicação no Negócio

Este dashboard pode ser utilizado por empresas para:

* Identificar regiões mais lucrativas
* Reduzir prejuízos operacionais
* Direcionar investimentos de forma estratégica
* Apoiar decisões de expansão

## 🚀 Conclusão

A análise regional permite uma visão clara do desempenho de vendas, ajudando empresas a focarem em regiões mais rentáveis e melhorarem seus resultados.
